# 🆘 ResQNet — AI Disaster Relief Planner
**Final Year Project | Python Flask + AI + Real Maps**

---

## 📦 Installation & Run

### Step 1: Install Python packages
```bash
pip install flask requests
```

### Step 2: Run the app
```bash
cd ResQNet
python app.py
```

### Step 3: Open in browser
```
http://localhost:5000
```

---

## 🌟 Features

| Feature | Description |
|---------|-------------|
| 🏠 Home | Landing page with live stats & hero section |
| 🚨 Alerts | Real-time disaster alerts with severity levels |
| 🗺️ Map | Interactive Leaflet.js map with earthquake & shelter data |
| 🏕️ Shelters | Shelter capacity finder with map integration |
| 🤖 AI Chat | AI assistant for disaster guidance (24/7) |
| 📖 Guide | Step-by-step safety guide for all disasters |
| 🆘 SOS | Emergency contacts modal (112, 108, 101...) |

## 🎓 Project Info
- **Framework:** Python Flask
- **Maps:** Leaflet.js (OpenStreetMap + CartoDB Dark)

- **Design:** Responsive CSS + animations
- **Data:** USGS real-time API + simulated shelter data
